Tātai

Tātai was a project undertaken by the part 2 software engineering cohort in 2017. The design brief was to create a lunix-based application with the purpose of helping users with pronounciation of Te Reo numbers, as well as testing basic arithmetic skills. The application features voice recognition software which can detect Te Reo numbers 1-99, as well as user tracked statistics, custom question input lists, and a simple, easy to user graphical user interface.
