Projects

Tātai

Tātai was a project undertaken by the part 2 software engineering cohort in 2017. The design brief was to create a linux-based application with the purpose of helping users with pronounciation of Te Reo numbers, as well as testing basic arithmetic skills. The application features voice recognition software which can detect Te Reo numbers 1-99, as well as user tracked statistics, custom question input lists, and a simple, easy to user graphical user interface.

BusterPong

BusterPong is a lightweight, easy-to-use Java implementation of the classic game Pong. The project was really a way for me to reinforce the object oriented practices I had acquired throughout the university year in the context of a personal project. I took the project as far as I thought necessary before I began BusterPong_Beta, which will be an extension of the application started from scratch, coming soon.
